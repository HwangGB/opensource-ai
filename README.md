# 2025년 오픈소스 생태계 내 생성형 AI의 통합

보조자에서 자율적 운영체제로의 진화 — AI가 이끄는 오픈소스의 새로운 패러다임

## 개요

2025년 12월 현재, 오픈소스 소프트웨어(OSS) 생태계는 생성형 AI의 도입으로 인해 근본적인 구조적 변화를 겪고 있습니다. 이 프로젝트는 해당 변화를 정리한 문서를 웹페이지로 제작한 것입니다.

## 주요 내용

1. **인지적 거버넌스의 자동화** - Dosu, Claude Action 등을 활용한 이슈 분류 및 동적 문서화
2. **운영의 지능화** - AI 프로젝트 매니저와 리스크 예측
3. **에이전트 개발자** - Claude Code를 이용한 코드 리뷰 및 수정
4. **MCP 표준화** - Model Context Protocol과 멀티 에이전트 시스템
5. **추론의 민주화** - GitHub Models와 무료 API 활용

## 파일 구조

```
├── index.html    # 메인 웹페이지
```
